# HarmandeepGportfolio

A highly immersive, visually stunning, and interactive graphic design portfolio website for Harmandeep Singh.

## 🚀 Technologies Used
- **Frontend**: Semantic HTML5, CSS3 Custom Properties (CSS variables), Vanilla JavaScript.
- **Visuals & Assets**: Custom SVG brand icons (Adobe Photoshop, Adobe Illustrator, Canva, Figma, CorelDRAW).
- **Animations**: CSS transitions, custom float & keyframe animations.
